Formats des fichiers images
==========================
BMP : bitmap (ancien format déconseillé)
ICO : Microsolft Icon
GIF : Graphic Interchange Format (petit à petit remplacé par PNG)

PNG : Portable Network Graphics (transparence gérée , 
        qualité correcte si Zoom , pratique pour capture ecran)

APNG : Animated PNG (avec animations possibles comme gif animés)

JPEG (.jpg, .jpeg , .jfif ) : Joint Photographic Expert Group image
  --> très bon niveau de compression mais perte de qualité si Zoom
  NB: variante ".pjpg ou .pjpeg" affichage progressif (image un peu flou 
  au départ puis de plus en plus nette/précise au fur et à mesure du téléchargement)
  extension pas toujours significative , beaucoup de .jpg sont des 
 .jfif(jpeg file interchange format), ...

SVG : Scalable Vector Graphic (cas particulier de Xml) : images vectorielles
     pratique pour diagrammes , graphiques , ...

TIFF : Tagged Image File Format (image brute , pas compréssée)

WebP : Web Picture Format (format récent , pas encore beaucoup utilisé )
       plus performant que jpeg (sans perte , meilleur compression, ...)
